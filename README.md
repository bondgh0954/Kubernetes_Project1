<h1>Demo Project</h1>

<h2>Project Description</h2>
Setup local K8s cluster with minkube
Deploy MongoDB and MongoExpress with configuration and credentials extrated into configMap and Sercret
<br />


<h2>Technologies used</h2>

- <b>Kubernetes</b> 
- <b>Docker</b>
- <b>MongoDB</b>
- <b>Mongo Express</b>
